# /note — 빠른 메모

지나가는 생각·발견·리마인더를 DB 에 즉시 던져 넣는다. (DB 미초기화면 먼저 `python bin/ws.py init`)

## 매핑

| 의도 | 명령 |
|---|---|
| 기록 | `python bin/ws.py note "메모 내용" [--tag 태그]` |
| 최근 조회 | `python bin/ws.py note [--limit N]` |

## 가이드

- 사용자가 "적어둬 / 메모 / 기억해둬" 라고 하면 본문을 그대로 `note` 로 넣는다.
- `--tag` 는 선택 — 주제가 뚜렷하면 한 단어로(예: `--tag payroll`), 애매하면 생략.
- 인자 없이 `note` 만 부르면 최근 메모를 보여준다("방금 뭐 적었지").
- **메모는 임시 상태(state)** 다. 하나의 지식으로 굳으면 `wiki/*.md` 로 승격한다(지식은 플랫 마크다운).
  - 승격이 필요해 보이면 사용자에게 "이건 wiki 로 옮겨둘까요?" 하고 물을 수 있다.
- 결정(왜 이렇게 했는지)은 `/note` 가 아니라 `/decide` 로 남긴다.
